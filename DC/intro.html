INTRODUCTION

No one ever said learning code was an easy task, but it’s certainly doable! Speciality with some good resources and help. Let’s get one thing straight, remember everyone is learning at their own pace, and you can’t know it all, so it’s okay to ask for help.

All great questions! Troubleshooting code can be difficult at first, but once you start understanding the tools at your disposal, troubleshooting will be a breeze!
