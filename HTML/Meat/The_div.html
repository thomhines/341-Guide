the div

The div tag is used for defining a section of your document. With the div tag, you can group sections of your HTML elements together and format them with CSS. This is similar to when you group objects within illustrator. Divs helps simplify layouts. How? By allowing you to keep relative and like objects together while positioning items, and will help later when you are focusing on creating classes and styling elements within these groups. We will talk about that later though.

To help visualize this idea, I have created two divs on the right. Just like the <html> and <body> tags the <div> tag has opening and closing tags that wrap around content. The content inside the divs is now considered grouped together by a “parent element.”

“Div” is such a vague term, and when you have multiple divs (trust me you will), it’s nice to be able to label your divs with specific titles by using ID’s or classes. Now, wait, what’s an ID? What’s a Class?

The role of an ID is to specify a unique element within an HTML document, hints the word unique. To use an ID you must understand these rules: Must contain at least one character, must not contain any space characters, all values are case-insensitive, and can not be repeated. Once you use an ID on an element once, you can never use that name again. The ID is uniquely connected with that element.

Classes work in the more global sense. They can be reused over and over again throughout the document.

As shown on the right, I have added a class of “div2” to the bottom div.

It’s important to note that you can have divs at multiple levels or divs within divs. This can get confusing quickly if you don't structure your code the right way. It can quickly be hard to read, that’s why I highly recommend intending.

What we have is three divs to the right. The top one is unchanged, but the bottom one (”div2”) now has a div within itself. I’ve also added an ID to the div inside “div2” as an example of the use of an ID. So what does that mean? Well... at the basic level, you have a group within a group. At the moment the use of layering div may not seem useful, but it will be a real skill to reference when working with CSS.
