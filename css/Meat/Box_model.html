The box model

Imagine this, that every element in web design is a rectangular box because it is. The size of every element is determined by a combination of factor: The height and width of the content, the border, padding, and the margin around the element. WOW, that’s a lot to take in, so let’s break it down one by one to help simplify it.

Let’s start with the simplest of those factors, the height, and width of the content. Think of a picture, the height and width are determined by the ratio of the photo and how many pixels it has. In web design elements develop their width and height through the same philosophy. For example, each line of type takes up a given amount of height depending on the type size, x-height and leading.

Next is the border. Think of the border like the stroke around a line or object in Illustrator. The technical definition is an individual border property (in values) in a single place in the style sheet. The border can be designed in multiple ways such as changing the color, style (solid, dashed, etc.), thickness, or a get creative with a mixture of styles.

The remaining values involved with the box model are padding and margin, and these can easily be the most difficult aspects of the model.

Padding provides your content a cushion from the edge of the of the border of the object (this is showcased on the right). This is handy for text within a box, or button. Padding values can be applied equally on all four sides, or you can use unique values to each side, set in numbers or percentages, to fit your needs.

Margin, on the other hand, pushes objects away from the object you are styling. This helps create negative space around the object you are designing. Setting a margin can help, and is the best way, to position items on your page. Like padding, values can be applied equally on all four sides or you can apply unique values to each side, set in numbers or percentages, to fit your needs.

The margin property has a unique ability called “margin auto,” it works like magic! And is incredibly useful. What margin auto does is ensures your object is perfectly centered within a set width container. This can come in handy when you start to worry about responsive design and is the easiest way to center objects.
