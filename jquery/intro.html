INTRODUCTION

So... would you like your website to change and evolve based off events? For example make something change by the click of a button on your site? like make something change size or color based on that click? Welcome to your new best friend, JavaScript and jQuery.

jQuery is a cross-platform JavaScript library designed to simplify to make it easier to navigate a document, select DOM elements, create animations, handle events, and develop Ajax applications. Simply, it helps you create cool shit quickly with less code.
